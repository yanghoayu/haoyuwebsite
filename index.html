Projects

Enhancing Cloud Detection with Explainable Machine Learning: A Study on FY-4B GIIRS Observations. (2024.2 - Present)

Cloud detection is essential for assimilating meteorological satellite observations and inverting atmospheric parameters. This project aims to develop a machine learning algorithm for cloud detection using FY-4B GIIRS observations. By utilizing machine learning explainable tools, the importance of different channels in cloud detection is analyzed, and a macro cloud parameter inversion model is established based on channel features to obtain cloud mask and cloud height parameters within the field of view. The algorithm demonstrates superior computational speed and detection accuracy compared to operational products.


Urban Complex Wind Field Reconstruction with Wind Lidar. (2022.09 - 2024.4)

Wind lidar is a remote sensing instrument known for its compact size and high temporal-spatial resolution. Due to its outstanding capability in fine wind field detection, wind lidar is often used for detecting complex urban wind fields. However, beam blockage caused by obstacles critically affects the quality of lidar data. To reconstruct wind fields blocked by obstacles, a wind field dataset is built to train the GANs model. The sensitivity of reconstruction effectiveness under different shielding conditions is studied to understand how shielding influences the reconstruction.
